# sudoku-game

## 项目介绍

大一学生纯兴趣依靠AI开发，一款基于网页纯前端开发的单机数独小游戏，无需安装浏览器点开即玩。

## 功能/玩法说明

最新版本 **v4.0** 具有以下功能：

- **主题切换** - 多种主题风格可选
- **关卡模式** - 挑战不同关卡
- **查看纪录** - 查看游戏历史记录
- **自选难度新游戏模式** - 可选择不同难度开始新游戏

其中**地狱难度**适合稍专业一点的数独选手选择。

## 开发环境

- **开发环境**：记事本
- **技术栈**：HTML + CSS + JavaScript
- **运行环境**：Edge浏览器
